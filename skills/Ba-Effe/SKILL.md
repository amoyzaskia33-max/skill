# Ba-Effe Skill - Effervescence Design Assistant

## Purpose

This skill helps you contribute to **Ba-Effe**, the design repository for **Effervescence**, the annual cultural fest of IIIT Allahabad.

**Theme:** Rangara (रंगारा) - A celebration of 70s-80s Bollywood cinema aesthetics

## When to Use

Use this skill when:
- Working on festival design tasks (posters, banners, merchandise, passes)
- Need guidance on 70s-80s Bollywood design aesthetics
- Contributing to OpenCode '25 design challenges
- Creating branding & visual identity for cultural events
- Working on UI/UX concepts for festival apps
- Creating motion graphics & 3D visuals

## Design Theme: Rangara

The design direction is inspired by **70s-80s Bollywood**:

### Key Aesthetics
- **Hand-painted film posters** with bold typography
- **Warm sepia tones** and technicolor dreams
- **Dramatic, colorful, and cinematic** energy
- **Masala cinema** spectacle
- **Vintage 1970s** romantic aesthetics
- **Disco era** vibrancy

### Visual Elements
- Hand-painted billboards
- Bold, dramatic typography
- Warm color palettes (sepia, orange, red, gold)
- Technicolor effects
- Rangeen (colorful) magic

## Contribution Workflow

### 1. Understand Issue Types

| Type | Claim Required? | Time Limit | Points |
|------|----------------|------------|--------|
| **Open-for-All (OFA)** | No | Flexible | Lower |
| **Competitive** | No | Flexible | Higher (best PR wins) |
| **First-Come** | Yes - comment to claim | 2 days | Standard |

### 2. Getting Started

```bash
# Fork and clone
git clone https://github.com/your-username/Ba-Effe.git
cd Ba-Effe

# Create your folder
mkdir -p your-username/issue-number
```

### 3. Folder Structure

For each issue, create:
```
your-username/
├── design-files/     # Source files (Figma, PSD, AI, etc.)
├── exports/          # Final outputs (PNG, SVG, MP4, etc.)
└── README.md         # Design explanation
```

### 4. Pull Request

- Title format: `Issue: #<issue_number>`
- Include design rationale in PR description
- Export files in appropriate formats
