# React Native Boilerplate 2025 - Modern Mobile App Starter

## Purpose

This skill provides a modern React Native boilerplate with the latest features including Expo, TypeScript, and Supabase authentication for quickly starting mobile app development.

## When to Use

Use this skill when:
- Building cross-platform mobile apps
- Need React Native starter template
- Implementing Supabase authentication
- Using Expo for development
- Creating apps with TypeScript
- Building MVP quickly

## Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | React Native 0.76+ |
| **Development** | Expo |
| **Language** | TypeScript |
| **Backend** | Supabase |
| **Auth** | Supabase Auth |
| **Navigation** | React Navigation |
| **State** | Zustand/Context API |
| **UI** | NativeWind/Tamagui |

## Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env

# Start Expo
npx expo start

# Run on device
npx expo run:android
npx expo run:ios
```
