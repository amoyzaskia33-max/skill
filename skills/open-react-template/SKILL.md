# Open React Template - Free SaaS Landing Page

## Purpose

This skill provides expertise in using the Open React Template, a free open-source landing page template by Cruip, designed for SaaS products, startups, and web applications.

## When to Use

Use this skill when:
- Building SaaS landing pages
- Creating startup websites
- Need modern React template
- Developing product showcases
- Building app landing pages
- Creating marketing pages

## Tech Stack

| Layer | Technology |

| **Styling** | Tailwind CSS |

## Quick Start

```bash
# Clone template
cd open-react-template
npm install

# Start development
npm run dev

# Build for production
npm run build
```

## For AI Assistants

When helping with landing page projects:

## Best Practices

- **Clear value prop** - Above the fold
- **Strong CTAs** - Multiple conversion points
- **Social proof** - Testimonials, logos
- **Fast loading** - Optimize images
- **Mobile first** - Responsive design
