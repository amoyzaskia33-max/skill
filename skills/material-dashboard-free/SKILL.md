# Material Dashboard Free - Material Design Admin Template

## Purpose

This skill provides expertise in using Material Dashboard, a free Material-UI based admin dashboard template featuring modern design, responsive layouts, and essential dashboard components.

## When to Use

Use this skill when:
- Building Material Design dashboards
- Need free admin template
- Creating modern admin panels
- Using Material-UI components
- Building SaaS dashboards
- Developing analytics platforms

## Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | React / Vue / Angular / HTML |
| **UI Library** | Material-UI (MUI) |
| **Styling** | SCSS, CSS Modules |
| **Charts** | Chart.js |
| **Icons** | Material Icons |
| **Routing** | React Router / Vue Router |

### Dashboard Features

| Feature | Description |

### Material Components

- 📊 **Cards** - Stats, info, charts
- 🔘 **Buttons** - Raised, flat, FAB
- 📝 **Forms** - Inputs, selects, checkboxes
- 📋 **Tables** - Sortable tables
- 🔔 **Alerts** - Notifications
- 🗂️ **Navigation** - Drawers, menus
- 📅 **Date Pickers** - Calendar inputs
- 🎨 **Theme** - Light/dark modes

## Quick Start

```bash
# Install dependencies
npm install

# Start development
npm run dev

# Build for production
npm run build
```
