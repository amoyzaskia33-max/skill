## Installation Guide

### 1. Upload Plugin

```
1. Download fs-poster-wordpress-auto-poster-scheduler-7.6.2.zip
2. WordPress Admin → Plugins → Add New → Upload Plugin
3. Choose the zip file and click "Install Now"
4. Activate the plugin
```

### 2. Connect Social Accounts

```
1. Go to FS Poster → Accounts
2. Click "Add Account" for each platform
3. Authorize with OAuth
4. Configure default settings
```

### 3. Configure Auto-Posting

```
1. FS Poster → Settings
2. Enable auto-post for post types
3. Set default message format
4. Configure URL shortener (optional)
```
