# Tabler Dashboard - Modern HTML Admin Template

## Purpose

This skill provides expertise in using Tabler, a premium-quality free HTML admin dashboard template featuring modern design, clean aesthetics, and extensive UI components built on Bootstrap.

## When to Use

Use this skill when:
- Building modern admin dashboards
- Need clean, professional design
- Creating SaaS admin panels
- Using Bootstrap framework
- Building analytics dashboards
- Developing CRM systems

## Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | Bootstrap 5 |
| **CSS** | SCSS, Custom CSS |
| **JS** | Vanilla JS, Bootstrap JS |
| **Icons** | Tabler Icons (open source) |
| **Charts** | ApexCharts, Chart.js |
| **Build** | Gulp, npm scripts |

## Quick Start

```bash
# Install via npm
npm install @tabler/core

# Or download from GitHub
# Include CSS and JS

# Start development
npm run dev

# Build for production
npm run build
```

### HTML Example

```html
<!DOCTYPE html>
<html>
<head>
  <title>Tabler Dashboard</title>
  <link rel="stylesheet" href="tabler.min.css">
</head>
<body>
  <div class="page">
    <!-- Sidebar -->
    <aside class="navbar navbar-vertical"></aside>
    
    <!-- Header -->
    <header class="navbar navbar-expand-md"></header>
    
    <!-- Content -->
    <div class="page-wrapper">
      <div class="page-body"></div>
    </div>
  </div>
  
  <script src="tabler.min.js"></script>
</body>
</html>
```
