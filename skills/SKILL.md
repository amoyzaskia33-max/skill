## 🔧 Cara Menggunakan

### Pattern Aktivasi Skill

1. **Identifikasi task** - Apa yang ingin dikerjakan?
2. **Cek skill yang relevan** - Lihat daftar di atas
3. **Baca SKILL.md** - Pahami panduan skill
4. **Ikuti instruksi** - Terapkan dalam task Anda

### Contoh Penggunaan

| Task | Skill yang Digunakan |
|------|---------------------|
| "Buat portfolio developer" | `developer-folio` atau `developer-portfolio-said` |
| "Belajar React dari nol" | `react-roadmap` |
| "Persiapan interview front-end" | `fe-interview-questions` |
| "Setup e-commerce marketplace" | `shopnest-ecommerce` |
| "Automate browser tasks" | `browser-use` |
| "Belajar full-stack development" | `fullstack-roadmap` |
| "Design poster festival" | `ba-effe` |
| "Plan & execute development" | `superpowers` |
| "Implement security best practices" | `security-guide` |
| "Learn by building clones" | `build-your-own-x` |
