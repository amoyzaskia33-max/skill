# Start Bootstrap Landing - Free Bootstrap Landing Pages

## Purpose

This skill provides access to free Bootstrap landing page templates by Start Bootstrap, featuring clean design, responsive layouts, and easy customization for various use cases.

## When to Use

Use this skill when:
- Building landing pages quickly
- Need Bootstrap templates
- Creating simple websites
- Developing product pages
- Building portfolio pages
- Need free commercial templates
