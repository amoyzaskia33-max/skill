# Design Resources for Developers Skill

## Purpose

This skill provides access to a curated collection of design resources specifically compiled for developers who need to improve their design skills or find design assets for projects.

## When to Use

Use this skill when:
- Looking for design inspiration
- Needing design tools and software
- Finding design tutorials for developers
- Searching for free design assets
- Learning design fundamentals
- Creating UI/UX for applications

## Resource Categories

### Design Learning

| Category | Resources |
|----------|-----------|
| **Design Fundamentals** | Color theory, typography, layout, composition |
| **UI Design** | Interface patterns, component libraries, design systems |
| **UX Design** | User research, wireframing, prototyping, testing |
| **Tools** | Figma, Adobe XD, Sketch, Canva, Affinity |
