# AdminLTE Template - Most Popular Admin Dashboard

## Purpose

This skill provides expertise in using AdminLTE, the most popular free and open-source admin dashboard template based on Bootstrap 5, with extensive components, plugins, and customization options.

## When to Use

Use this skill when:
- Building admin dashboards quickly
- Need Bootstrap-based template
- Want extensive plugin ecosystem
- Creating enterprise admin panels
- Building CRM/ERP systems
- Need responsive design out-of-box

## Quick Start

```bash
# Via CDN (simplest)
# Include CSS and JS in HTML

# Or install via npm
npm install admin-lte

# Import in your project
import 'admin-lte/dist/css/adminlte.min.css';
import 'admin-lte/dist/js/adminlte.min.js';
```

### HTML Structure

```html
<!DOCTYPE html>
<html>
<head>
  <title>AdminLTE Dashboard</title>
  <!-- AdminLTE CSS -->
  <link rel="stylesheet" href="adminlte.min.css">
</head>
<body class="hold-transition sidebar-mini">
  <div class="wrapper">
    <!-- Navbar -->
    <nav class="main-header navbar"></nav>
    
    <!-- Main Sidebar -->
    <aside class="main-sidebar"></aside>
    
    <!-- Content -->
    <div class="content-wrapper"></div>
    
    <!-- Footer -->
    <footer class="main-footer"></footer>
  </div>
  
  <!-- AdminLTE JS -->
  <script src="adminlte.min.js"></script>
</body>
</html>
```
