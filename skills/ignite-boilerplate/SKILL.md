# Ignite Boilerplate - Production-Ready React Native

## Purpose

This skill provides expertise in using Ignite, a battle-tested React Native boilerplate from Infinite Red, featuring code generators, testing setup, and proven architectural patterns.

## When to Use

Use this skill when:
- Building production React Native apps
- Need proven architectural patterns
- Using code generators for productivity
- Implementing comprehensive testing
- Following best practices
- Scaling mobile teams

## Quick Start

```bash
# Create new app with Ignite
npx ignite-cli@latest new MyApp

# Or use existing boilerplate
cd ignite-boilerplate
npm install

# Start development
npm start

# Run on device
npm run android
npm run ios
```

## Code Generators

```bash
# Generate component
npx ignite-cli generate component Button

# Generate screen
npx ignite-cli generate screen Login

# Generate model
npx ignite-cli generate model User

# Generate app icon
npx ignite-cli generate app-icon ./icon.png
```

## For AI Assistants

When helping with Ignite projects:

1. **Understand architecture** - MVVM or similar pattern
2. **Use generators** - Don't create files manually
3. **Follow conventions** - Ignite has specific patterns
4. **Testing strategy** - Unit + E2E with Detox
5. **State management** - Understand MST patterns

## Best Practices

- **Use generators** - Consistent file structure
- **Write tests** - Critical flows with Detox
- **Type everything** - Full TypeScript coverage
- **Component driven** - Reusable UI components
- **API layer** - Standardized API handling
