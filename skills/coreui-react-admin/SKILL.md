# CoreUI React Admin - Enterprise Admin Template

## Purpose

This skill provides expertise in using CoreUI, a free enterprise-grade React admin template featuring professional components, modern layouts, and extensive customization options.

## When to Use

Use this skill when:
- Building enterprise dashboards
- Need React admin template
- Creating professional admin panels
- Using CoreUI components
- Building SaaS applications
- Developing analytics platforms

## Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | React 18+ |
| **UI Library** | CoreUI Components |
| **Styling** | SCSS, CSS Modules |
| **State** | Redux Toolkit |
| **Routing** | React Router v6 |
| **Charts** | Chart.js, C3.js |
| **Icons** | CoreUI Icons, Font Awesome |

## Quick Start

```bash
# Clone template
cd coreui-react-admin
npm install

# Start development
npm run dev

# Build for production
npm run build
```
