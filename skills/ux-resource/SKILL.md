# UX Resource Skill - UI/UX Design Reference Hub

## Purpose

This skill provides access to a curated collection of UI/UX design resources, articles, tools, and references for improving design skills and finding inspiration.

## When to Use

Use this skill when:
- Looking for UI/UX design inspiration
- Need references for design patterns
- Searching for design tools and resources
- Learning about UX best practices
- Finding articles on design methodology
- Exploring color palettes, typography, or layout references

## How to Access Resources

1. **Browse the repository** - Navigate through categorized links
2. **Search by topic** - Find resources relevant to your design challenge
3. **Follow curated links** - Access external articles, tools, and galleries

## For AI Assistants

When helping with design tasks:

1. **Understand the design need** - What is the user trying to design?
2. **Reference relevant resources** - Point to specific design patterns or articles
3. **Suggest tools** - Recommend appropriate design tools for the task
4. **Provide inspiration** - Share relevant design galleries or examples

## Design Best Practices

- **User-Centered Design** - Always start with user needs
- **Accessibility** - Design for all users (WCAG guidelines)
- **Consistency** - Maintain design system consistency
- **Iteration** - Design, test, refine
- **Mobile-First** - Design for smaller screens first
